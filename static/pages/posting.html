ヘルプ

トリップコード

トリップコードを使用するには、名前のところに#を付けてください。の後に トリップコードのパスワード

サーバーはパスワードにハッシュを適用し、ハッシュ化されたパスワードを表示します。

セキュアトリップコードを利用するには、名前に##を付け、パスワードを設定します。セキュアトリップコードは、パスワードが一致した場合にのみ使用されるランダムなテキストを生成し、表示します。

ロールシグネチャー

投稿時にグローバルや板の役割を表示したい場合は、トリップコードのように名前に#rsを大文字小文字を区別せずに入れることができます。これにより、トリップコードを使用しないようにすることができます。

また、アカウントで常にロール署名を使用するルールを有効にすることができます。

マークダウン

投稿する際に、そのテキストを強化するツールをいくつか使用することができます。HTMLです。

緑色の文字

行頭に > を付けると、テキストを緑色で表示します。 新しい行を見つけるまで


例：

>こんにちは！ :^) -> >こんにちは！ :^)


引用

投稿に >> と数字を書くと、投稿中の掲示板にこの投稿へのリンクが作成され、javascriptが有効になっていれば投稿プレビューでツールチップを追加してくれます。


例：

>>50 -> >>50


板リンク

>>と書いて、スラッシュの間に板名を入れると、板へのリンクが作成されます。


例：

>>>/映画/ -> >>>/映画/


クロスクオート

また、他の掲示板の投稿を引用することも可能です。掲示板名の後に投稿番号を追加するだけです。


例：

>>>/映画/50 -> >>>/映画/50


タグ

[spoiler]ゲーム[/spoiler] -> ゲーム

**これも効く** -> これも効く

==SRZ BIZNIZ== -> SRZ BIZNIZ

''イタリック'' -> イタリック

'''太字''' -> 太字

__テキストに下線を引く__ -> テキストに下線を引く

~~取り消し線のテキスト~~ -> 取り消し線のテキスト

[aa]日本語文字用

[code]板所有者によって有効にされている場合はタグ


Urls

httpまたはhttpsで始まるurlを投稿すると、自動的にリンクされます。


例：

http://bot.net -> http://bot.net
